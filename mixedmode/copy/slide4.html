Entities

Entities are the building blocks of a Mixed Mode application. By default, every entity contains three components: size, opacity, and transform. However additional components can be added. By adding and manipulating these components, we can generate animations, display media, and use physics in our application.

To create an entity, simply call the addChild method on the main context or on another entity. After inspecting the code to the right, head over to the next slide to learn about displaying content with a surface component.
