Positioning

As mentioned previously, every entity has a size, opacity, and transform component. For this lesson, we will focus on the transform component and learn how to use it to position a surface within an application.

In order to modify a component, you must first access it using the getComponent method. Since we are concerned with positioning an entity, we will need to access the translate method on the transform component. Check out the code to the right to see how this is done.
