Entity Component System

Famo.us Mixed Mode is based on an Entity Component System. It may sound complex, but in practice it is really simple. Everything in a Mixed Mode application is connected through an entity.

Entities carry several attributes or components and attach them to the main stage. Check out at the tree like structure pictured to the right. This is the Famo.us scene graph and the basic layout to every Mixed Mode app. Notice how entities are child nodes extending from a single context.
