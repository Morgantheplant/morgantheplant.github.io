THE BEST SELLER WORKSHEET


Problem solved – Sifts through 60K+ rows of data, extracts sales info and ranks "Best Sellers" by class.

ABOUT: After building the SKU EXPLODER I decided to find other ways to make myself more efficient at work. One particularly grueling task was pulling together our "Best Sellers" lists at the beginning of each week. Even with conditional formatting and a number of other tricks this would still take each of us several hours for weekly reports and sometimes the entire day for the month end recaps. This worksheet helped me reduce that time significantly and build more accurate reports for all of my assigned categories.

FORMAT: Each week two sales reports (Reg price and Outlet) are exported from Ecometry as Excel files with 60K lines of sales data each. Since the files are so big and accessed via our local network, I built the Best Sellers Worksheet to update with a copy and paste instead of linking to an external worksheet. This helps speed up the calculations and prevents Excel from crashing when changing class or category info.



The reg price Weekly Sales Report is pasted into a special tab in columns A – V. In the next column over, column W, a function checks if the class from the TOTAL Style (the pivot total for each color ie. TOTAL Style+C: A3M 01 03FLM YE) matches the class listed on the Best Seller table (M 01 03 in image above) and displays it if it does. In the next two columns X and Y, if column W has a SKU listed (checks if length is > 1 character), then the total units sold for the week and total retail sales are also listed.

Then, based on what’s displayed in columns W-Y (numbered 1-3 in the image below) an array formula ranks, matches, and lists the top 15 SKUs for Past Week Total Units Sold and Past Week Total Sales in 2 separate tables. Additional SKU info (description, on hand units, etc.) is then added to the left of these tables using a simple VLOOKUP to columns A-V.



In order to limit re-typing data, the info from the Best sellers tables are then re-listed on a separate tab in the traditional format of the Best Sellers Report.



The final result above lists the top 10 styles in the following format (see image and below):

  1. Category - This Data validated cell can be changed and will update all class info in the worksheet based on this cell.
  2. Override feature - This was added so you can "skip" certain styles (GWP's or on consignemnet goods) or rank top ten by either sales or units. Both Best seller tables list top 15 styles to help compensate with additional info if a style is skipped.
  3. Description - Pulled from the best seller table and uses a function to trim off last word to eliminate color.
  4. Image area - This is where an image is placed. Often used in conjunction with the IMAGE GRABBER.
  5. Sales info - Simple concatenation and calculation for WOS based on Best Seller table info.
  6. Color - Found on SKU and looked up against a color table.

Even though this sheet was eventually replaced by a more robust pivot table based report, it was very useful in the interim. I had a lot of fun building it and learned a lot that I later incorporated into my other worksheets.
